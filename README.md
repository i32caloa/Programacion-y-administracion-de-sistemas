# Programación y Administración de Sistemas (PAS)

Este repositorio contiene las prácticas de la asignatura **Programación y Administración de Sistemas**, perteneciente al **2º curso** del Grado en Ingeniería Informática de la **Universidad de Córdoba (UCO)**. 

El código y los documentos aquí incluidos han sido desarrollados durante el curso académico **2025/2026**.

## Sobre la Asignatura

El enfoque principal de las prácticas es la **programación en Shell** para la automatización de tareas y la administración de sistemas operativos basados en Unix/Linux. A través de los distintos ejercicios se abordan conceptos de gestión de procesos, usuarios, permisos, archivos de texto y servicios del sistema.

## Estructura del Repositorio

El trabajo está dividido en 4 prácticas. Cada una de ellas cuenta con su propio directorio independiente, donde se encuentran los scripts desarrollados y los archivos de prueba necesarios para su ejecución:

* **`/p1`**: Programación de la shell
* **`/p2`**: Expresiones regulares para la programación de la shell
* **`/p3`**: [Descripción opcional de la tercera práctica].
* **`/p4`**: [Descripción opcional de la cuarta práctica].

Dentro de cada carpeta se incluye el código fuente (archivos `.sh`) y la estructura de directorios o ficheros que requiere cada ejercicio en particular.

## Requisitos y Uso

Para probar y ejecutar los scripts almacenados en este repositorio, se necesita un entorno Linux/Unix o un subsistema compatible (como WSL) que cuente con el intérprete de comandos Bash.

Antes de ejecutar cualquier script, es necesario otorgarle permisos de ejecución. Puedes hacerlo con el siguiente comando:

```bash
chmod +x nombre_del_script.sh
